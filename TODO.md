🎛️ 2. РЕЖИМЫ РАБОТЫ СИСТЕМЫ
2.1 Режим 1: RPM динамический (существующий)

    CAN ID: 0x003

    Описание: Прием RPM значений 4 колес каждые 1 мс

    Данные: 4 × uint16_t (FL, FR, RL, RR)

    Индикация: LED мигает 2 Гц

    Приоритет: Стандартный режим по умолчанию

2.2 Режим 2: Фиксированная частота

    CAN ID: 0x004

    Описание: Установка точной фиксированной частоты на таймеры

    Особенности:

        Python вычисляет оптимальные PSC/ARR

        Показывает фактическую частоту и ошибку

        Однократная установка (не динамическая)

    Индикация: LED мигает 1 Гц

    Сохранение: Сохраняется в EEPROM при команде

2.3 Режим 3: ШИМ режим

    CAN ID: 0x004

    Описание: Установка частоты и скважности

    Параметры: Частота (Гц), скважность (0-100%)

    Индикация: LED мигает 0.5 Гц

    Применение: Для тестирования драйверов

2.4 Режим 4: Следование аналоговому сигналу

    CAN ID: 0x004

    Описание: Таймеры повторяют частоту внешнего сигнала

    Источник: Вход компаратора/ADC

    Индикация:

        2 Гц - сигнал присутствует

        0.5 Гц - сигнал отсутствует

    Особенность: Автоматическая подстройка частоты

2.5 Режим 5: Выключено

    CAN ID: 0x004

    Описание: Все таймеры остановлены

    Индикация: LED постоянно выключен

    Назначение: Безопасный режим

2.6 Специальные режимы

    Режим загрузки (BOOT): 5 Гц мигание

    Режим ошибки: 10 Гц мигание

📡 3. ПРОТОКОЛ CAN КОМАНД
3.1 Существующий формат (ID 0x003)
text

Назначение: Передача RPM значений
Длина: 8 байт
Формат:
  Байт 0-1: RPM переднее левое (uint16_t BE)
  Байт 2-3: RPM переднее правое
  Байт 4-5: RPM заднее левое
  Байт 6-7: RPM заднее правое

3.2 Новый формат команд (ID 0x004)
text

Длина: 8 байт
Байт 0: Код команды:
  0x00 - Выключить все выходы
  0x01 - Перейти в RPM режим
  0x02 - Установить фиксированную частоту
  0x03 - Установить ШИМ
  0x04 - Режим следования аналоговому сигналу
  0x05 - Сохранить текущий режим в EEPROM
  0x06 - Заводской сброс EEPROM
  0x07 - Запросить статус системы
  
Байт 1: Маска каналов:
  Бит 0: TIM1 (FL)
  Бит 1: TIM2 (FR) 
  Бит 2: TIM3 (RL)
  Бит 3: TIM4 (RR)
  0xFF: Все каналы

Байт 2-5: Параметр 1 (зависит от команды):
  Для команд 0x02, 0x03: Частота в Гц (uint32_t LE)
  Для других: 0x00000000

Байт 6-7: Параметр 2:
  Для команды 0x03: Скважность 0-100% (uint16_t LE)
  Для команды 0x02: Общий PSC (uint16_t LE) или 0xFFFF для авто
  Для других: 0x0000

3.3 Формат ответа статуса (ID 0x006)
text

Длина: 8 байт
Байт 0: Текущий режим (0-4)
Байт 1: Маска активных каналов
Байт 2-3: Частота TIM1/100 Гц (uint16_t)
Байт 4-5: Частота TIM2/100 Гц (uint16_t)
Байт 6-7: Резерв

void update_status_led(void) {
    static uint32_t last_toggle = 0;
    uint32_t interval_ms;
    
    switch(current_mode) {
        case MODE_BOOT:        interval_ms = 100; break;  // 5 Гц
        case MODE_RPM:         interval_ms = 250; break;  // 2 Гц
        case MODE_FIXED_FREQ:  interval_ms = 500; break;  // 1 Гц
        case MODE_PWM:         interval_ms = 1000; break; // 0.5 Гц
        case MODE_ANALOG:
            interval_ms = analog_signal_present ? 250 : 1000;
            break;
        case MODE_ERROR:       interval_ms = 50; break;   // 10 Гц
        default:               interval_ms = 0;  // Постоянно
    }
    
    if(interval_ms > 0 && HAL_GetTick() - last_toggle > interval_ms) {
        HAL_GPIO_TogglePin(STATUS_LED_GPIO_Port, STATUS_LED_Pin);
        last_toggle = HAL_GetTick();
    }
}

void save_current_state_to_eeprom(void) {
    eeprom_config_t config;
    config.magic_number = 0x55AA55AA;
    config.saved_mode = current_mode;
    config.saved_frequency = target_frequency;
    config.saved_duty = pwm_duty;
    config.saved_channels = active_channels_mask;
    config.crc32 = calculate_crc32(&config, sizeof(config)-4);
    
    HAL_FLASHEx_DATAEEPROM_Program(FLASH_TYPEPROGRAMDATA_WORD, 
                                   EEPROM_BASE_ADDR, 
                                   (uint32_t)&config, 
                                   sizeof(config));
}

7.2 Восстановление при старте
c

void load_state_from_eeprom(void) {
    eeprom_config_t config;
    memcpy(&config, (void*)EEPROM_BASE_ADDR, sizeof(config));
    
    if(config.magic_number == 0x55AA55AA && 
       config.crc32 == calculate_crc32(&config, sizeof(config)-4)) {
        // Восстанавливаем состояние
        current_mode = config.saved_mode;
        target_frequency = config.saved_frequency;
        apply_saved_state();
    } else {
        // EEPROM пустой или поврежден
        current_mode = MODE_RPM;  // Режим по умолчанию
    }
}
